# CAMPUSCONNECT
roadmap
## About


Student communities in colleges often face confusion and delays while booking labs due to manual permission processes and lack of clear availability information.

This project aims to simplify lab booking by providing a **web-based system** to check lab availability and manage permission approvals in one place.

---

## Problem Statement

Lab booking and permission approvals in colleges are handled manually, leading to miscommunication, scheduling conflicts, and wasted time for students, faculty, and authorities.

---

## Solution

A simple web application that allows:

- Community leads to request labs
- Faculty to review and approve requests
- Authorities to give final approval
- Everyone to track request status clearly

---

## Key Features (MVP)

- Role-based login
- Lab availability view
- Permission request & approval flow
- Request status tracking

---

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: SQLite / Firebase
