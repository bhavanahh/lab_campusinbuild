# CampusConnect

CampusConnect is a web-based application designed to simplify **lab availability checking and permission management** for college communities.
---

## Problem Statement
In colleges, student communities often face confusion and delays while booking labs due to manual permission processes and lack of clear availability information. Coordination between community leads, faculty, and college authorities is time-consuming and untracked.

---


## Solution
Campusync provides a **single platform** that connects:
- Community Leads
- Faculty / Lab In-charges
- College Authorities

The system allows users to check lab availability, request permissions digitally, and track approval status step by step.

---

## Key Features (MVP)
- Role-based login (Community Lead, Faculty, Authority)
- Lab availability view
- Lab request submission
- Approval workflow
- Request status tracking
